# 🎵 Music Player

A modern and responsive Music Player built using **HTML, CSS, and JavaScript**. The application provides an intuitive interface for playing, pausing, and navigating songs with smooth audio controls.

## 🚀 Features

- ▶️ Play and Pause music
- ⏭️ Next & Previous track navigation
- 🎵 Playlist support
- 📊 Interactive progress bar
- ⏱️ Seek through audio tracks
- 🔊 Volume control
- 📱 Responsive design
- 🎨 Clean and modern user interface
